OBJECTIVE: Cart/wishlist global state + bubble effect + spotlight cards

SHOP CONTEXT (lib/ShopContext.tsx):
  Key behavior per spec:
  - addToCart: if product already in cart, INCREMENTS quantity (no duplicate)
  - addToCartFromWishlist: adds to cart but KEEPS the wishlist item
  - placeOrder: clears ONLY cart. Wishlist survives the order.
  - Navbar badges now show live counts from context

GLOW CARD:
  CSS pointer-tracking technique using CSS custom properties --x/--y
  set on every pointermove event. The ::before/::after pseudo-elements
  read these values via radial-gradient positioning. The result is a
  glowing border that appears to track the cursor globally across the page.
  Applied to FeaturedProducts and Shop page cards.

BUBBLE BUTTON:
  On click, spawns 8-13 bubble particles at the exact click position
  with random angles and velocities. Each bubble is a framer-motion
  span that animates to scale 0 as it travels outward. A shockwave
  ring also expands from the click point. The existing btn-bubble CSS
  class is preserved — the BubbleButton component and useBubbleEffect
  hook ADD on top of existing effects.

ANIMATED SOCIAL ICONS:
  Plus button slides RIGHT (not up) to reveal 4 social icons behind it.
  Icons animate from blurred/rotated to clear/straight with spring physics.
  Integrated in Footer bottom bar.
